## 当前 Demo 目标

当前版本只追求：

- 可运行；
- 可试玩；
- 可验证核心体验；
- 可展示核心玩法；
- 可展示主要 UI 流程；
- 可快速迭代。

当前 Demo 重点验证：

- 玩家是否理解核心操作；
- 玩家是否理解主要目标；
- 玩家是否能完成一次核心循环；
- 玩家是否能看懂关键 UI 信息；
- 关键反馈是否足够清楚；
- 这个方向是否值得继续投入 Godot Demo 或完整开发。

当前版本不追求：

- 最终美术品质；
- 完整商业化；
- 完整后端；
- 完整关卡数量；
- 完整上线品质。

## 商业模式方向

Demo 阶段可以不验证商业化。

如需要，可在后续阶段评估：

- 买断制；
- 广告激励；
- 轻度内购；
- 章节 / 内容包；
- 订阅；
- 其他。

商业化不得破坏核心体验。
