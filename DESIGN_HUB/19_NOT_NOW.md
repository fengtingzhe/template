# 19_NOT_NOW：当前阶段暂不做清单

## 定位

本文件用于记录当前阶段明确不做的内容。

它的作用是防止 AI 过度发散、过早开发大型系统，或把当前 Demo 做成完整商业项目。

---

## 通用暂不做内容

当前阶段默认不做：

```text
完整账号系统；
完整后端；
正式支付系统；
正式广告 SDK；
完整商业化系统；
复杂多人在线；
大型开放世界；
复杂运营后台；
复杂数据埋点平台；
完整发布流程；
正式商用美术；
正式音频资源；
大规模内容生产；
复杂自动化测试平台；
从 0 创建复杂 Unity 工程，除非明确进入 v2.x。
```

---

## v0.x / Web Demo 阶段暂不做

```text
不做完整 Unity 项目；
不做完整存档系统；
不做复杂资源管线；
不做复杂动画系统；
不做复杂后端；
不做正式商业化；
不做大规模关卡库；
不做复杂编辑器后台；
不做与当前验证目标无关的功能。
```

---

## v1.x / Unity 源码学习阶段暂不做

```text
源码未跑通前不做大规模改造；
不绕过许可证；
不替换大量资源；
不升级 Unity 大版本，除非明确确认；
不重写核心架构；
不把外部源码直接当作最终项目；
不把不明来源资源用于商用。
```
