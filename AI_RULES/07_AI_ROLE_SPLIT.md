# 07_AI_ROLE_SPLIT：AI 角色分工

## ChatGPT

定位：

```text
制作人 / 总策划 / 需求统筹 / 产品复审
```

适合任务：

- 项目定位；
- 玩法方案；
- 系统设计；
- 数值思路；
- 任务拆解；
- 验收标准；
- 复盘总结。

---

## Codex

定位：

```text
主程 / 架构师 / 代码审核员 / Bug 修复负责人
```

适合任务：

- 搭建工程结构；
- 编写核心代码；
- 审核 DeepSeek 提交；
- 修复报错；
- 整合功能；
- 维护项目可运行状态。

---

## DeepSeek

定位：

```text
模块开发工程师 / 批量实现助手 / 配置生成助手
```

适合任务：

- 普通功能模块；
- UI 小组件；
- 配置表；
- 文案；
- 批量数据；
- 简单工具函数。

不适合任务：

- 决定整体架构；
- 修改核心系统；
- 合并主干；
- 最终代码审核。

---

## 人类制作人

负责：

- 最终方向判断；
- 核心玩法拍板；
- 体验判断；
- 优先级排序；
- 是否接受 AI 建议；
- 关键设计取舍；
- 最终 Demo 验收。
